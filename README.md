# BUS2 Test

Projeto Flutter de mostra, salva ou delete do banco offile usuários obtidos atraves da chamada dessa [API](https://randomuser.me/api/) 

## Escopo

Projeto tem como objetivo demonstrar minhas habilidades tecnicas utilizando Flutter. 

## Como Rodar

Copie o projeto em sua maquina, abra-o em um editor de código de sua preferencia, abra o terminar e use o ``` flutter run ```
